# Stopwatch Application

A sleek and professional stopwatch web application built with HTML, CSS, and JavaScript. It features precise time tracking, lap recording with split times, keyboard shortcuts, and persistent lap storage using localStorage. The design incorporates modern UI elements like glassmorphism, gradients, and animations for an engaging user experience.

## Features

- *Start, Pause, Reset, and Lap Functions*: Control the stopwatch with intuitive buttons.
- *High-Precision Timing*: Displays hours, minutes, seconds, and milliseconds (updated every 10ms).
- *Lap Tracking*: Records laps with total time and split differences; laps are displayed in a scrollable list.
- *Persistent Data*: Laps are saved in the browser's localStorage, persisting across page reloads.
- *Keyboard Shortcuts*:
  - Spacebar: Start/Pause
  - Ctrl/Cmd + R: Reset
  - L: Record Lap
- *Responsive Design*: Adapts to mobile and desktop screens.
- *Accessibility*: Includes ARIA attributes for screen reader support.
- *Aesthetic UI*: Minimalist neutral gradient background, animated elements, and hover effects inspired by modern web designs.

## Installation

1. Clone the repository:
   
   git clone https://github.com/Chandan3466/SCT_WD_02.git
   
2. Navigate to the project directory:
   
   cd stopwatch-app
   
3. Open index.html in your preferred web browser.

No additional dependencies or build steps are required, as it's a pure vanilla HTML/CSS/JS project.
